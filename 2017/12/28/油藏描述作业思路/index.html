油藏描述作业思路

  • 作业思路

提要

由于我是采用回忆的方法来总结经验，诸多考虑不到的地方，请多包含。

您可以直接或通过邮件的方式咨询。

注意事项

  1. petrel 软件在用于计算 砂泥岩厚度，及其比值（这些值多后期沉积微相的划分）时，可事半功倍。但petrel 2014的软件安装费时费力（并不推荐）。
  2. 老师极力推荐手绘/coreldraw绘图。这也是我此后得知的，因而我本人不再更改报告的任何细节；谨以此告诫大家。
  3. 在处理数据时，特别是针对22个数据单，有些繁杂；我极力推崇大家采用恰当的Excel函数打辅助，并从此迷上自动化。如INDIRECT函数等。如果能利用课外时间学习少量的VBA知识，也是极好的。
  4. 精力充沛的同学，可以在完成所有任务项之余，再用手清绘一遍，定会直捣黄龙。所以，不要再问到底什么方法好的问题。

思路

步骤 说明 工具方法 备注
1.油藏井位十字剖面线 油藏剖面线、闭合高 用 Coreldraw 在纵横向将油藏所有井位连接、从纵横向计算同一井位的高差（高差为0，即为闭合 [ 不必验证所有 ]） 前提
2. 油层细分对比 4-1小层的识别、划分（砂泥岩层）、砂岩厚度计算、制表汇总 依循1中的剖面线将井位排列并识别4-1层（依据三高泥岩段）、砂泥岩划分（半幅法）、petrel划分自动厚度计算/Coreldraw比例尺手动读厚、Excle制表统计 基础
3.等值线图 砂厚、孔隙度、渗透率、含油饱和度的等值线图绘制 根据由2中的砂岩厚度、及所给公式将孔隙度、渗透率、含油饱和度分别计算，归置四张表内，用surfer/python绘制等值线图即可 高潮
4.沉积微相图 依据所给出的基本该油藏基本沉积微相表，找相同 在考虑地质规律，结合砂泥厚度比下，结合砂岩等厚图，用Coreldraw的图层管理快捷地完成沉积微相图的绘制工作 过渡
5.油藏剖面图 总结考虑四个等值线图及沉积微相图，确定油水界面 选取1中的某一条剖面线，标定出油水界面海拔高即可。推荐用Coreldraw绘制该图（用petrel是被要扣分的，别问我为什么知道） 尾声
